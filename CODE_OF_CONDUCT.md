# Code of Conduct

Moxie follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version: be kind, be honest, assume good faith, no harassment of any
kind. This project exists to earn trust with people's money — the same standard
applies to how we treat each other.

Report unacceptable behaviour privately to the maintainers (see SECURITY.md for
the private contact route). Reports are handled confidentially.
