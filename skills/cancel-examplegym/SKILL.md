# Cancel ExampleGym membership

Follows the `SKILL.md` convention used across the OpenClaw / Hermes ecosystem
(agentskills.io). Moxie proposes these steps; **you approve each one** before
anything is sent.

## Steps

1. Find the account email used at sign-up.
2. Email `member-services@examplegym.com` with the account email and the last 4 digits of the card on file.
3. Decline any retention offer; request **written confirmation** of cancellation.
4. If no confirmation arrives within 14 days, escalate to a card chargeback (attach the Moxie receipt as evidence).

## Notes

- Never share full card numbers or passwords.
- Only act on the user's own account.
